# Ecolight
EcoLight es una aplicación nativa de Android diseñada para ayudar a los amantes de las plantas a encontrar el lugar perfecto para sus compañeras verdes. Utilizando el sensor de luz ambiental (TYPE_LIGHT) integrado en los dispositivos móviles, la app mide la luz en tiempo real y determina si las condiciones son óptimas para una especie específica.
